C# 变量及作用域

发表于 2018-04-16 | 分类于 学习
字数统计: 304 字 | 阅读时长 ≈ 1 分钟

1.变量

  变量是用以存储特定类型的数据，可以根据需要随时改变变量中所存储的数据值。

变量三要素：

  • 类型
  • 名称
  • 值

  变量按作用域来划分的话可以分为全局变量和局部变量。全局变量在整个程序中引用；而局部变量只可以在程序中的有限部分被引用。

2.作用域

  作用域是指可访问程序中的字段、属性、方法等的区域。

  • 类的字段所处的作用域等同于该字段所属类所在的作用域。
  • 局部变量的作用域仅限于声明它的方法的或循环体内部,以大括号{}为界。

C# 基础知识回顾（一）

1.循环

  循环由迭代语句创建。 根据循环终止条件，迭代语句可导致多次执行嵌入式语句。 除非遇到跳转语句，否则这些语句按顺序执行。

1.1 do

  • do…while 循环是在循环的尾部检查它的条件。
  • do…while 循环与 while 循环类似，但是 do…while 循环会确保至少执行一次循环。

1.2 while

  因为 while 表达式的测试在每次执行循环之前开始，所以 while 循环执行零次或多次。 这不同于 do 循环，该循环执行一次或多次。

1.3 foreach，in

  foreach 语句针对实现 System.Collections.IEnumerable 或 System.Collections.Generic.IEnumerable 接口的数组或集合中的每个元素重复一组嵌入语句。 foreach 语句用于循环访问集合以获取所需信息，但不用于添加或删除源集合中的项，以避免不可预知的副作用。 如果需要添加或删除源集合中的项，请使用 for 循环。

1.4 for

  通过使用 for 循环，可以重复运行一个语句或语句块，直到指定的表达式的计算结果为 false 为止。 这种类型的循环可用于循环访问数组，以及事先知道循环要在其中进行循环访问的次数的其他应用程序。

2.分支

2.1 if-else

  一个 if 语句 后可跟一个可选的 else 语句，else 语句在布尔表达式为假时执行。

  • 在 C# 中，嵌套 if-else 语句是合法的，这意味着您可以在一个 if 或 else if 语句内使用另一个 if 或 else if 语句。

2.2 switch

  switch 是一个选择语句，它根据与匹配表达式匹配的模式，从候选列表中选择单个开关部分进行执行。

  • switch 语句包含一个或多个开关部分。 每个开关部分包含一个或多个 case 标签，后接一个或多个语句。
  • switch 语句中可以包含任意数量的开关部分，每个开关部分可以具有一个或多个 case 标签,但是，任何两个 case 标签不可包含相同的表达式。
  • 每个 case 语句定义一个模式，如果它与匹配表达式相匹配，则会导致执行其包含的开关部分。 所有版本的 C# 都支持常量模式。 其余模式从 C# 7 开始支持。

3.值类型

  值类型分为结构和枚举。

  • 结构类型又分为：
  • 数值类型
  • 整型类型
  • sbyte  -128~127  8 位带符号整数
  • byte  0~255  无符号的 8 位整数
  • char  U+0000 到 U+ffff  Unicode 16 位字符
  • short  -32,768 到 32,767  有符号 16 位整数
  • ushort  0 到 65,535  无符号 16 位整数
  • int  -2,147,483,648 到 2,147,483,647  带符号的 32 位整数
  • uint  0 到 4,294,967,295  无符号的 32 位整数
  • long  -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807  64 位带符号整数
  • ulong  0 到 18,446,744,073,709,551,615  无符号 64 位整数
  • 浮点类型
  • float  ±1.5e−45 到 ±3.4e38  7 位精度
  • double  ±5.0e−324 到 ±1.7e308  15-16 位
  • 小数
  • decimal  (-7.9 x 1028 - 7.9 x 1028) / (100 - 1028)  28-29 个有效位
  • bool
  • 用户自定义结构
  • struct 类型是一种值类型，通常用来封装小型相关变量组，例如，矩形的坐标或库存商品的特征。

  enum 关键字用于声明枚举，一种包含一组被称为枚举数列表的已命名常数的不同类型。

  • 默认情况下，第一个枚举数具有值 0，并且每个连续枚举数的值将增加 1。
  • 每个枚举类型都有一个基础类型，该基础类型可以是除 char 外的任何整型类型；有 byte、sbyte、short、ushort、int、uint, long 或 ulong。

4.引用类型

  C# 中有两种类型：引用类型和值类型。 引用类型的变量存储对其数据（对象）的引用，而值类型的变量直接包含其数据。

  • 对于引用类型，两种变量可引用同一对象；因此，对一个变量执行的操作会影响另一个变量所引用的对象。
  • 对于值类型，每个变量都具有其自己的数据副本，对一个变量执行的操作不会影响另一个变量（in、ref 和 out 参数变量除外）。

可通过下列关键字声明引用类型：

  • class
  • interface
  • delegate

c#内置引用类型：

  • dynamic
  • object
  • string

-在通过 dynamic 类型实现的操作中，该类型的作用是绕过编译时类型检查。改为在运行时解析这些操作。

5.集合

  通常我们使用泛型集合以获得类型安全的直接优点而无需从基集合类型派生和实现特定类型的成员，同时也避免了对集合元素进行装箱操作。

常见泛型集合类型：

  常见集合类型：

类 描述
Dictionary 表示基于键进行组织的键/值对的集合。
List 表示可按索引访问的对象的列表。 提供用于对列表进行搜索、排序和修改的方法。
Queue 表示对象的先进先出 (FIFO) 集合。
SortedList 表示基于相关的 IComparer 实现按键进行排序的键/值对的集合。
Stack 表示对象的后进先出 (LIFO) 集合。

常见非泛型集合类型：

类 描述
ArrayList 表示对象的数组，这些对象的大小会根据需要动态增加。
Hashtable 表示根据键的哈希代码进行组织的键/值对的集合。
Queue 表示对象的先进先出 (FIFO) 集合。
Stack 表示对象的后进先出 (LIFO) 集合。
